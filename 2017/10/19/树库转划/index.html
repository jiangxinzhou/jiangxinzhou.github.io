结论:

  • 大规模CTB5上我们的版本比张月师姐的性能好(差的不多)

  • multi-task大规模数据上，CTB5的比baseline提升1.0+, 但是CDT上比baseline降低了，不知是不是数据的比例给的不对，CDT数据集太大了？

2017.10.19 看多树库融合的论文

paper name : exploiting multiple treebanks for parsing with quasi-synchronous grammars

一些图片：

我的理解：

  • QG features 就是在basic parsing features的特种模板中结合了TP的类型。这样，在目标parser创建树时，就能用到用到源端的信息。

  • QG features 是用于学习原标注和目标标注之间的联系。
